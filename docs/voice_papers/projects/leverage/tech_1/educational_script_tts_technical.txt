Definición de palanca (leverage): Es decir, Mecanismo por el cual un input pequeño o constante produce un output mayor <break time="1.5s"/>
Ejemplo en mecánica clásica: <break time="1.0s"/>
- Aplicación de fuerza descendente en el lado izquierdo de una palanca para levantar una masa de 1 kilogramo en el lado derecho <break time="0.5s"/>
- Aumento del tamaño de la palanca permite un factor de multiplicación de 3× <break time="1.5s"/>
Tres tipos de palanca según Naval Rob: <break time="1.0s"/>
1. Trabajo humano: <break time="0.5s"/>
   - Uso de miles de trabajadores con input individual constante que genera mayor output agregado <break time="1.0s"/>
2. Capital: <break time="0.5s"/>
   - Inversión de 200 mil dólares para comprar una propiedad de un millón de dólares mediante 800 mil dólares de préstamo; amplifica el retorno cuando la propiedad se valoriza <break time="1.0s"/>
3. Código y medios: <break time="0.5s"/>
   - Escribir código para una aplicación que otorga valor a cada nuevo usuario sin trabajo adicional <break time="0.5s"/>
   - Publicación de una conferencia en línea que genera valor incremental por cada visualización <break time="1.5s"/>
